Gemini API 金鑰設定

本工具完全在您的瀏覽器本地運行。您的 API 金鑰將儲存在本機的 localStorage 中，直接發送請求至 Google Gemini API，絕不會上傳到任何第三方伺服器。

後端已啟用 GEMINI_API_KEY 環境變數，若無特殊需求，您無需在此處設定金鑰。

💡 如何取得免費的 API 金鑰？

  1. 前往 Google AI Studio 。
  2. 登入您的 Google 帳號。
  3. 點擊左上角的 "Get API key"。
  4. 點擊 "Create API key"，選擇一個專案或新建專案，複製產生的金鑰。
  5. 將金鑰貼到上方的輸入框中，並點擊儲存。
